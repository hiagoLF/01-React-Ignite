# Executar e Testar

- Requisito: NodeJs

```bash
# Depois de clonar o projeto, navegue até a pasta raiz dele instale as dependências com o comando...
yarn install
# ou
npm i

# Para executar o projeto rode o comando
yarn dev
# ou
npm run dev

# Para executar os testes...
yarn test
# ou
npm run test
```
